Name Appearance Example Description
HTML Element <p>Paragraph</p> p { } Selects all <p> elements
id <p id="specificname">Unique</p> #specificname { } Selects an element with the id 'specificname'. Must be unique
class <img class="portrait" src="george.jpg" alt="portrait of george"> .portrait { } Selects all elements with the class 'portrait'
attribute <a href="google.com">Visit Google<> [href] { } Selects all elements with an href attribute
attribute value <a href="https://en.wikipedia.org/wiki/The_Third_of_May_1808">Interesting Wikipedia article<> [href*="wikipedia"] { } Selects all elements with a href attribute containing 'wikipedia'
